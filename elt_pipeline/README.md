# elt_pipeline

Pada pipeline ELT, data mentah dimuat terlebih dahulu ke dalam Data Warehouse tanpa preprocessing. Transformasi, pembersihan data, serta feature engineering dilakukan langsung di dalam database menggunakan SQL. Pendekatan ini menunjukkan pemanfaatan query engine database untuk analitik skala besar dan dibedakan secara jelas dari pipeline ETL berbasis Python.
